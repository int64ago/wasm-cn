Inside the browser

  • Better execution for languages and toolkits that are currently cross-compiled to the Web (C/C++, GWT, â€¦).
  • Image / video editing.

  • Peer-to-peer applications (games, collaborative editing, decentralized and centralized).
  • Music applications (streaming, caching).
  • Image recognition.

Outside the browser

  • Game distribution service (portable and secure).
  • Server-side compute of untrusted code.
  • Server-side application.
  • Hybrid native apps on mobile devices.

How WebAssembly can be used
